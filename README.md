# Testing AI Code Generation

Today, 21st February 2026, we asked GitHub CoPilot integrated in Visual Studio Code to create a game.

We prompted it to create a game like Super Mario with three sibblings. Also, it was supposed to be a browser game and it to be based on node.js and using webpack.

Initial outcome of the first commit was ready after less than half an hour.

# What the game is about.

Simple browser jump-and-run game featuring the Hintsch siblings Paul, Lena and Bruno.

# Getting started

1. Make sure node.js is installed.

1. Install dependencies:
    ```bash
    npm install
    ```

1. Start development server:
    ```bash
    npm start
    ```

The dev server opens at http://localhost:3000 and displays a selection screen.

Controls
- Left/Right arrows or A/D to move
- Space / Up / W to jump
